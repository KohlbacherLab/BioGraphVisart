BioGraphVisart is a web-based tool to interactively visualize networks, especially with biological background. It is written in javascript based on Cytoscape.js.

The input is a graphml-file (graphml-example) or sif-file (sif-example). In addition to nodes and edges (genes and interaction types, respectively), GraphML can store information such as expression values as node attributes (more on GraphML Format). SIF cannot store additional information (more on SIF Format).

For direct Comparison of network graphs the node overlap can be calculated and be displayed in a heatmap. Also a merged network graph of two selected network graphs can be generated. The input are multiple graphml-files.
